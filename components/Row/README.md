# @doc-blocks/row

Welcome to my `Row` component.

## Installation

```sh
npm i @doc-blocks/row
# or with yarn
yarn add @doc-blocks/row
```

## Usage

**Example:**

```jsx
import { Row } from "@doc-blocks/row";

<Row>
  <div style={{ height: 60, width: "100%", background: "black" }} />
  <div style={{ height: 60, width: "100%", background: "black" }} />
  <div style={{ height: 60, width: "100%", background: "black" }} />
</Row>;
```
